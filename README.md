# Gerador de Senhas

## Como iniciar o projeto ❔
- Clone o repósito
- Abra a pasta do projeto no VS Code
- Abra o terminal e digite 'npm run dev'
  * É necessário ter o NodeJS instalado
- Execute o arquivo index.html
